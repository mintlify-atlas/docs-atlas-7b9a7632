# Introduction

Vuetify is a no design skills required UI library with beautifully handcrafted Vue components. Everything you need to create amazing applications is at your fingertips.

## Why Vuetify?

Vuetify provides a massive API that supports any use-case, from small prototypes to enterprise applications. Here's what makes Vuetify stand out:

### Highly customizable

Extensive customization options with SASS/SCSS variables, default configuration, and blueprints. Tailor your application's look and feel to match your brand perfectly.

### Responsive by default

The default configuration of Vuetify components is responsive, allowing your application to adapt seamlessly to different screen sizes without extra configuration.

### Powerful theme system

A sophisticated color system that makes it easy to style your application with a consistent color palette. Switch between light and dark modes effortlessly.

### Vite support

Smaller bundle sizes with automatic tree-shaking. Vuetify is optimized for modern build tools, ensuring your applications load fast.

### Long-term support

6 months of long-term support for major releases, giving you confidence in production deployments.

### Global reach

Built-in internationalization with 42+ supported languages, making it easy to create applications for users worldwide.

## Browser support

Vuetify supports all modern browsers, including Safari 13+ (using polyfills). Components are designed for a minimum width of 320px.

Learn more about [browser compatibility](/browser-support).

## Open source and MIT licensed

Vuetify is a MIT licensed project developed and maintained by a dedicated core team. The framework is free to use in both personal and commercial projects.
