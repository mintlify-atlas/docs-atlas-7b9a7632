# Vuetify Ecosystem

The Vuetify ecosystem includes a wide range of tools and resources to help you build amazing applications with Vue.js.

## Quick Start

To create a new Vuetify project, choose your package manager and run one of the following commands:

<CodeGroup>
```bash pnpm
pnpm create vuetify
```

```bash yarn
yarn create vuetify
```

```bash npm
npm create vuetify@latest
```

```bash bun
bun create vuetify
```
</CodeGroup>

For more information on how to get started, such as using Nuxt or Laravel, check out the official [Installation guide](https://vuetifyjs.com/getting-started/installation/).
