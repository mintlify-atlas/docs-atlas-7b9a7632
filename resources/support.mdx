# Getting Support

Need help with Vuetify? We offer multiple channels for getting support and connecting with the community.

## Discord Community

Our Discord server is the best place to get help, share feedback, and connect with other Vuetify developers.

<Card title="Join Discord" icon="discord" href="https://community.vuetifyjs.com/">
  Join our massive and inclusive Discord server where you can ask questions, share feedback, and connect with other Vuetify developers.
</Card>

### What You Can Do on Discord

- Ask questions and get help from the community
- Share your projects and get feedback
- Discuss features and best practices
- Connect with the Core Team
- Stay updated on the latest news and releases

<Note>
For help and support questions, please use our [Discord community](https://community.vuetifyjs.com). The GitHub issue tracker is **exclusively** for bug reports and feature requests.
</Note>

## Reporting Issues

Found a bug or want to request a feature? Use our Issue Generator to create a detailed report.

<Card title="Issue Generator" icon="bug" href="https://issues.vuetifyjs.com/">
  A web application for reporting bugs and issues with Vuetify, Documentation, or one of our other packages.
</Card>

### Before Reporting an Issue

Please make sure to:

1. Read the [Important Information](https://github.com/vuetifyjs/vuetify/blob/master/.github/CONTRIBUTING.md#important-information) in our contributing guide
2. Search existing issues to avoid duplicates
3. Use the Issue Generator to ensure all required information is included
4. Issues not conforming to the guidelines may be closed immediately

<Warning>
The issue list is **exclusively** for bug reports and feature requests. For general questions and discussions, please use the [Discord community](https://community.vuetifyjs.com/).
</Warning>
