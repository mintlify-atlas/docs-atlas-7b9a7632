# Roadmap

Learn about the future direction of Vuetify and what's coming next.

## Development Philosophy

Vuetify is committed to providing a no design skills required UI Library with beautifully handcrafted Vue Components. Our roadmap focuses on:

- Maintaining backward compatibility where possible
- Improving performance and bundle size
- Expanding component functionality
- Enhancing developer experience
- Supporting modern web standards

## Current Focus Areas

## Ongoing Improvements

### Theme System

Continuous enhancements to the powerful color system and theme customization:
- Extended color palette options
- Improved dark mode support
- Better theme switching capabilities
- Enhanced SASS/SCSS customization

### Internationalization

Expanding language support beyond the current 42+ supported languages and improving RTL (Right-to-Left) support.

### Browser Support

Vuetify supports all modern browsers, including Safari 13+ with polyfills. We continue to:
- Monitor browser compatibility
- Add support for new web standards
- Optimize for mobile browsers
- Ensure components work at minimum width of 320px

## Feature Requests

We actively track and consider community feature requests. To suggest new features:

1. Check existing [feature requests on GitHub](https://github.com/vuetifyjs/vuetify/labels/enhancement)
2. Use our [Issue Generator](https://issues.vuetifyjs.com/) to submit new requests
3. Participate in discussions on [Discord](https://community.vuetifyjs.com/)

## Long-term Support

Vuetify provides **6 months** of long-term support for major releases, including:

- Critical bug fixes
- Security patches
- Documentation updates
- Community support

Learn more about our [Long-term Support policy](https://vuetifyjs.com/introduction/long-term-support/).

## Platform Support

Vuetify aims to support a wide range of development environments:

- **Vue 3:** Full support with Composition API
- **Vite:** First-class support with optimized build tooling
- **Nuxt:** Official integration and support
- **Laravel:** Integration guides and tooling
- **SSR/SSG:** Server-side rendering and static site generation
