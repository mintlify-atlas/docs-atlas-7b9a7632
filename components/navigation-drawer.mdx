# VNavigationDrawer

The VNavigationDrawer component is used to display navigation links in a sidebar. It supports multiple display modes including permanent, temporary, and rail variants, with optional touch gestures and expand-on-hover behavior.

## Usage

```vue
<template>
  <v-navigation-drawer v-model="drawer">
    <v-list>
      <v-list-item
        v-for="item in items"
        :key="item.title"
        :to="item.to"
      >
        <template v-slot:prepend>
          <v-icon>{{ item.icon }}</v-icon>
        </template>
        <v-list-item-title>{{ item.title }}</v-list-item-title>
      </v-list-item>
    </v-list>
  </v-navigation-drawer>
</template>

<script setup>
import { ref } from 'vue'

const drawer = ref(true)
const items = [
  { title: 'Dashboard', icon: 'mdi-view-dashboard', to: '/dashboard' },
  { title: 'Account', icon: 'mdi-account', to: '/account' },
  { title: 'Settings', icon: 'mdi-cog', to: '/settings' },
]
</script>
```

## Props

<ParamField path="modelValue" type="boolean | null" default="null">
Controls the visibility of the navigation drawer.

```vue
<v-navigation-drawer v-model="drawer">
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="location" type="'start' | 'end' | 'left' | 'right' | 'top' | 'bottom'" default="'start'">
Sets the position of the navigation drawer.

```vue
<v-navigation-drawer location="end">
  <!-- Drawer on the right side -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="width" type="number | string" default="256">
Sets the width of the navigation drawer in pixels.

```vue
<v-navigation-drawer :width="300">
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="rail" type="boolean | null" default="null">
Condenses navigation drawer width to show only icons.

```vue
<v-navigation-drawer rail>
  <!-- Mini variant showing only icons -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="railWidth" type="number | string" default="56">
Sets the width of the drawer when in rail mode.

```vue
<v-navigation-drawer rail :rail-width="72">
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="expandOnHover" type="boolean" default="false">
When in rail mode, expands the drawer on hover.

```vue
<v-navigation-drawer rail expand-on-hover>
  <!-- Expands on hover -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="permanent" type="boolean" default="false">
Makes the drawer permanent (always visible, no overlay).

```vue
<v-navigation-drawer permanent>
  <!-- Always visible -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="temporary" type="boolean" default="false">
Makes the drawer temporary (overlays content, shows scrim).

```vue
<v-navigation-drawer temporary>
  <!-- Temporary overlay -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="persistent" type="boolean" default="false">
Keeps drawer open when clicking the scrim.

```vue
<v-navigation-drawer temporary persistent>
  <!-- Clicking scrim won't close -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="floating" type="boolean" default="false">
Makes the drawer float above the content without pushing it.

```vue
<v-navigation-drawer floating>
  <!-- Floating drawer -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="scrim" type="boolean | string" default="true">
Controls the scrim overlay. Can be a boolean or color string.

```vue
<!-- No scrim -->
<v-navigation-drawer :scrim="false">
  <!-- Content -->
</v-navigation-drawer>

<!-- Custom scrim color -->
<v-navigation-drawer scrim="rgba(0,0,0,0.8)">
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="image" type="string">
Specifies a background image for the drawer.

```vue
<v-navigation-drawer image="/path/to/image.jpg">
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="color" type="string">
Applies specified color to the drawer background.

```vue
<v-navigation-drawer color="primary">
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="touchless" type="boolean" default="false">
Disables touch gestures for opening/closing the drawer.

```vue
<v-navigation-drawer touchless>
  <!-- No touch gestures -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="disableResizeWatcher" type="boolean" default="false">
Disables the automatic opening/closing based on screen size.
</ParamField>

<ParamField path="disableRouteWatcher" type="boolean" default="false">
Disables automatic closing when route changes.
</ParamField>

<ParamField path="sticky" type="boolean" default="false">
Makes the drawer stick to the top of the viewport when scrolling.
</ParamField>

<ParamField path="elevation" type="number | string">
Sets the elevation (box-shadow) of the drawer.

```vue
<v-navigation-drawer elevation="8">
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="border" type="boolean | string | number">
Applies a border to the drawer.
</ParamField>

<ParamField path="rounded" type="boolean | string | number">
Applies border radius to the drawer.
</ParamField>

<ParamField path="tag" type="string" default="'nav'">
Specifies the custom tag to use for the root element.
</ParamField>

<ParamField path="theme" type="string">
Applies a specific theme to the drawer.
</ParamField>

<ParamField path="mobile" type="boolean | null">
Overrides the mobile breakpoint detection.
</ParamField>

<ParamField path="mobileBreakpoint" type="number | 'xs' | 'sm' | 'md' | 'lg' | 'xl' | 'xxl'">
Sets the breakpoint at which the drawer becomes temporary.
</ParamField>

<ParamField path="name" type="string">
Assigns a specific name for layout registration.
</ParamField>

<ParamField path="order" type="number | string" default="0">
Adjusts the order in which the drawer appears in the layout.
</ParamField>

<ParamField path="absolute" type="boolean" default="false">
Applies position: absolute to the drawer.
</ParamField>

<ParamField path="focusTrap" type="boolean" default="false">
Traps focus within the drawer when open.
</ParamField>

<ParamField path="openDelay" type="number | string">
Delay (in ms) before opening when using expand-on-hover.
</ParamField>

<ParamField path="closeDelay" type="number | string">
Delay (in ms) before closing when using expand-on-hover.
</ParamField>

## Events

<ParamField path="update:modelValue" type="(value: boolean) => void">
Emitted when the drawer visibility changes.

```vue
<v-navigation-drawer @update:modelValue="onDrawerChange">
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

<ParamField path="update:rail" type="(value: boolean) => void">
Emitted when the rail state changes (typically from expand-on-hover).

```vue
<v-navigation-drawer
  rail
  expand-on-hover
  @update:rail="onRailChange"
>
  <!-- Content -->
</v-navigation-drawer>
```
</ParamField>

## Slots

<ParamField path="default" type="slot">
The default slot for drawer content.
</ParamField>

<ParamField path="prepend" type="slot">
Slot for content at the top of the drawer (before default content).

```vue
<v-navigation-drawer>
  <template v-slot:prepend>
    <v-list-item title="User Name" subtitle="user@example.com">
      <template v-slot:prepend>
        <v-avatar color="primary">
          <v-icon>mdi-account</v-icon>
        </v-avatar>
      </template>
    </v-list-item>
  </template>
  
  <v-list>
    <!-- Navigation items -->
  </v-list>
</v-navigation-drawer>
```
</ParamField>

<ParamField path="append" type="slot">
Slot for content at the bottom of the drawer (after default content).

```vue
<v-navigation-drawer>
  <v-list>
    <!-- Navigation items -->
  </v-list>
  
  <template v-slot:append>
    <div class="pa-2">
      <v-btn block>
        Logout
      </v-btn>
    </div>
  </template>
</v-navigation-drawer>
```
</ParamField>

<ParamField path="image" type="slot">
Slot for custom background image content. Receives `{ image: string }` as slot props.

```vue
<v-navigation-drawer image="/bg.jpg">
  <template v-slot:image="{ image }">
    <v-img :src="image" gradient="to top, rgba(0,0,0,.7), rgba(0,0,0,.7)" />
  </template>
</v-navigation-drawer>
```
</ParamField>

## Examples

### Basic Drawer

```vue
<template>
  <v-app>
    <v-app-bar>
      <v-app-bar-nav-icon @click="drawer = !drawer"></v-app-bar-nav-icon>
      <v-toolbar-title>Application</v-toolbar-title>
    </v-app-bar>

    <v-navigation-drawer v-model="drawer">
      <v-list>
        <v-list-item
          v-for="item in items"
          :key="item.title"
          :to="item.to"
        >
          <template v-slot:prepend>
            <v-icon>{{ item.icon }}</v-icon>
          </template>
          <v-list-item-title>{{ item.title }}</v-list-item-title>
        </v-list-item>
      </v-list>
    </v-navigation-drawer>

    <v-main>
      <!-- Content -->
    </v-main>
  </v-app>
</template>

<script setup>
import { ref } from 'vue'

const drawer = ref(true)
const items = [
  { title: 'Home', icon: 'mdi-home', to: '/' },
  { title: 'About', icon: 'mdi-information', to: '/about' },

]
</script>
```

## Behavior Modes

### Temporary
- Overlays content
- Shows scrim (overlay)
- Automatically becomes temporary on mobile devices
- Closes on route change (unless disabled)

### Permanent
- Always visible
- No scrim
- Content adjusts to make space
- Does not close on route change

### Persistent
- Similar to temporary
- Doesn't close when clicking scrim
- Must be closed explicitly
